## Навбар

- При нажатии на логотип "VK Реклама" происходит переход на главную страницу (https://ads.vk.com/)
- При нажатии на кнопку "Новости" открывается страница новостей (https://ads.vk.com/news)
- При наведении курсора на "Обучение" выпадает меню с выбором из 4-х пунктов
- При наведении курсора на "Обучение" и выборе в выпадающем меню пункта "Полезные материалы" открывается страница полезных материалов (https://ads.vk.com/insights)
- При наведении курсора на "Обучение" и выборе в выпадающем меню пункта "Мероприятия" открывается страница мероприятий (https://ads.vk.com/events)
- При наведении курсора на "Обучение" и выборе в выпадающем меню пункта "Видеокурсы" происходит переход на сайт видеокурсов (https://expert.vk.com/catalog/courses/)
- При наведении курсора на "Обучение" и выборе в выпадающем меню пункта "Сертификация" происходит переход на сайт сертификаций (https://expert.vk.com/certification/)
- При нажатии на кнпоку "Кейсы" происходит переход на страницу кейсов (https://ads.vk.com/cases)
- При нажатии на кнопку "Форум идей" открывается страница форума идей (https://ads.vk.com/upvote)
- При нажатии на кнопку "Монетизация" открывается страница монетизации (https://ads.vk.com/partner)
- При нажатии на кнопку "Справка" открывается страница справки (https://ads.vk.com/help)
- При нажатии на кнопку "Перейти в кабинет" происходит переход на страницу VKID (https://ads.vk.com/hq/dashboard?requiredAccount=advert)
- Наведение на любую кнопку подсвечивает её

- При нажатии на ссылку "Документы" открывается страница документов (https://ads.vk.com/documents)
- При нажатии на ссылку "Обучение для бизнеса" происходит переход на страницу (https://expert.vk.com/)
- При нажатии на ссылку "Кейсы" происходит переход на страницу кейсов (https://ads.vk.com/cases)
- При нажатии на ссылку "Монетизация" открывается страница монетизации (https://ads.vk.com/partner)
- При нажатии на ссылку "Помощь" открывается страница справки (https://ads.vk.com/help)
- Кнопка "Перейти в кабинет" подсвечивается при наведении на неё курсора
- Все ссылки из раздела "РАЗДЕЛЫ" подсвечиваются при наведении курсора на них
- При нажатии на логотип "VK бизнес" происходит переход на страницу VK бизнеса (https://vk.company/ru/company/business/)
- При нажатии на логотип "VK" происходит переход страницу VK рекламы в VK (https://vk.com/vk_ads)
- При нажатии на логотип "OK" происходит переход на страницу VK рекламы в OK (https://ok.ru/group/64279825940712)
- При нажатии на логотип "Telegram" происходит переход на страницу сообщества VK рекламы в Телеграме (https://t.me/vk_ads)
- При нажатии на надпись языка "RU" или "EN" выпадает меню с выбором языка
- При нажатии на надпись языка "RU" или "EN" и выборе "English", страница становится на английском языке ("https://ads.vk.com/en")
- При нажатии на надпись языка "RU" или "EN" и выборе "Russian", страница становится на русском языке ("https://ads.vk.com/en") (не работает)
- При нажатии на сслыку "О кампании" происходит переход на страницу VK Company (https://vk.company/ru/)

## Списки пользователей

- При нажатии на кнопку "Загрузить список" справа вылезает окно с загрузкой списка
- Загрузить список
- Отменить загрузку списка по кнопке "Отмена"
- При нажатии на кнопку с троеточием выпдает меню с одной ссылкой
- При активации внешнего списка пользователей открывается окно с полем для ввода ключа
- При активации внешнего списка пользователей открывается окно с полем для ввода ключа. При неверном введении ключа, появляется сообщение "Ключ не найден"
- Поделиться списком
- Поиск списка пользователей
